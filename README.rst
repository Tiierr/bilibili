Bilibili live's barrage parser
==============================

I'm a small script that help you get bilibili live's barrage use aiohttp.

Install
-------

::

    sudo pip install bilibili

Usage
-----

::

    $ bilibili 679461
    正在进入房间...
    链接弹幕中...
    进入房间成功...
    链接弹幕成功...
    房间人数为 133
    悦悦77 送出了 1 个 辣条
    ......
    帅气可耐男神夜君 : 因吹思婷
    喜欢-看天 : UP 大气科学
    哔了高数 : 他总是在吃
    喜欢-看天 : 在读PHD
